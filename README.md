本教程将会一步一步引导大家从零开始搭建一个前端WebApp项目

涉及到的技术：

1. 怎样构建一个npm工程
2. git的克隆、拉取、推送、分支等
3. Vue、Axios、JRoll、JTaro等工具/库的使用
4. 怎样与后端对接接口
5. 字体图标的使用
6. 弹窗组件开发
7. ...
